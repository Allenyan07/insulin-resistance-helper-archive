🩺 胰岛素抵抗管理

🥗饮食要点
进食顺序：先喝汤 → 蔬菜 → 蛋白质 → 最后吃主食，能显著降低餐后血糖
餐盘比例：1/2蔬菜 + 1/4蛋白 + 1/4低GI主食，主食一拳头大小
绝对禁止：含糖饮料、甜点、勾芡浓汤、裹粉油炸食品
水果原则：每天不超过一拳头，两餐之间吃，不喝果汁。优选蓝莓草莓樱桃柚子
晚餐19:00前吃完，之后不进食。每天喝水1.5-2升
🏃运动要点
先抗阻后有氧：先做深蹲/弹力带等力量训练，再做快走/慢跑等有氧，燃脂效率更高
每周至少150分钟中高强度运动，分散到3-5天完成
从低强度起步，每周只增加5-10%，不追求一次到位。运动后30分钟内补充蛋白质
😴睡眠要点
23:00前入睡，保证7-8小时。周末不偏差超过1小时
睡前30分钟放下手机，蓝光抑制褪黑素。可用热水泡脚、轻柔拉伸助眠
🧘压力管理
腹式深呼吸：吸气4秒 → 憋气4秒 → 呼气6秒，每天2-3次，每次5分钟
降低完美主义：运动「动10分钟也算数」，饮食「这一餐比上一餐好一点就行」
